# Ansible Automation Roles

This repository contains a collection of Ansible roles for automating system configuration and setup on Linux systems.

The roles are designed to be reusable, modular, and easy to maintain. They can be used across different environments such as servers, virtual machines, and cloud instances.

## Current Roles

### Linux Hardening

This role applies basic security hardening to Linux systems, including:

* Disabling unnecessary services
* Securing SSH configuration
* Applying basic system security settings
* Configuring firewall rules

### Docker Installation

This role installs and configures Docker:

* Installs Docker Engine
* Enables and starts Docker service
* Configures user access to Docker
